파이썬(Python)은 1991년 귀도 반 로섬이 개발한 인터프리터 언어로, **간결하고 직관적인 문법** 덕분에 전 세계에서 가장 인기 있는 프로그래밍 언어 중 하나입니다. 데이터 분석, 웹 개발, 인공지능까지 활용 범위가 광대합니다. 이번 시리즈에서는 파이썬의 핵심 문법을 체계적으로 배워봅니다.

---

## 1. 파이썬의 특징

| 특징 | 설명 |
|------|------|
| **인터프리터 언어** | 코드를 한 줄씩 실행 → 즉각적인 결과 확인 가능 |
| **동적 타이핑** | 변수 선언 시 자료형을 명시하지 않아도 됨 |
| **간결한 문법** | 들여쓰기(Indent)로 블록 구조를 표현 |
| **풍부한 라이브러리** | pandas, numpy, sklearn 등 강력한 패키지 생태계 |
| **무료 & 오픈소스** | 누구나 언제 어디서든 무료 사용 가능 |

## 2. 변수(Variable)

변수는 데이터를 저장하는 **이름 붙은 공간**입니다.

```python
# 변수 선언과 대입
name = "김민준"      # 문자열
age = 25             # 정수
height = 175.5       # 실수
is_student = True    # 불린

# 여러 변수 동시 대입
x, y, z = 1, 2, 3
print(x, y, z)  # 1 2 3

# 변수 값 교환 (파이썬의 강력한 기능!)
a, b = 10, 20
a, b = b, a
print(a, b)  # 20 10
```

> [!TIP]
> 변수 이름은 **소문자와 언더스코어**를 사용하는 `snake_case` 방식이 파이썬 표준입니다.
> 예: `user_name`, `total_count`, `equipment_id`

## 3. 숫자형 자료형과 연산자

### 3.1 숫자 자료형

| 자료형 | 예시 | 설명 |
|--------|------|------|
| `int` | 1, -5, 0 | 정수 |
| `float` | 3.14, -1.5 | 실수 (소수점) |
| `complex` | 2+3j | 복소수 |

### 3.2 산술 연산자

```python
a, b = 10, 3

print(a + b)   # 13 (더하기)
print(a - b)   # 7  (빼기)
print(a * b)   # 30 (곱하기)
print(a / b)   # 3.333... (나누기 - 항상 float)
print(a // b)  # 3  (몫 - 정수 나눗셈)
print(a % b)   # 1  (나머지)
print(a ** b)  # 1000 (거듭제곱: 10³)
```

### 3.3 복합 대입 연산자

```python
x = 10
x += 5   # x = x + 5  → 15
x -= 3   # x = x - 3  → 12
x *= 2   # x = x * 2  → 24
x //= 5  # x = x // 5 → 4
```

## 4. 문자열(String)

### 4.1 문자열 만들기

```python
s1 = "Hello Python"
s2 = 'Python is fun'
s3 = """여러 줄의
문자열을 입력할 때 사용합니다."""
```

### 4.2 인덱싱과 슬라이싱

```python
text = "Python"
#       P y t h o n
#       0 1 2 3 4 5  (앞에서)
#      -6-5-4-3-2-1  (뒤에서)

print(text[0])    # 'P'
print(text[-1])   # 'n' (마지막 문자)
print(text[0:3])  # 'Pyt' (0~2번 인덱스)
print(text[::2])  # 'Pto' (2칸씩 건너뛰며)
```

### 4.3 주요 문자열 메서드

```python
s = "  Hello, Python!  "

print(s.strip())           # "Hello, Python!" (양쪽 공백 제거)
print(s.upper())           # "  HELLO, PYTHON!  "
print(s.lower())           # "  hello, python!  "
print(s.replace('o', '0')) # "  Hell0, Pyth0n!  "
print(s.split(','))        # ['  Hello', ' Python!  ']
print(len(s))              # 18 (문자열 길이)
print(s.count('l'))        # 2 (문자 개수)
print('Python' in s)       # True (포함 여부)
```

### 4.4 f-string (포맷 문자열, 가장 많이 씀!)

```python
name = "min oh"
score = 95.5

# f-string: f"...{변수}..." 형태
print(f"이름: {name}, 점수: {score}점")  # 이름: min oh, 점수: 95.5점
print(f"점수: {score:.1f}점")            # 소수점 1자리: 95.5점
print(f"100 + 200 = {100 + 200}")        # 중괄호 안에 계산식도 가능
```

## 5. 리스트(List)

순서가 있고 **수정 가능**한 자료구조입니다.

```python
# 리스트 생성
fruits = ['사과', '바나나', '딸기', '포도']
numbers = [1, 2, 3, 4, 5]
mixed = [1, 'Python', 3.14, True]  # 다양한 자료형 혼합 가능

# 인덱싱과 슬라이싱
print(fruits[0])   # '사과'
print(fruits[-1])  # '포도'
print(fruits[1:3]) # ['바나나', '딸기']
```

### 리스트 주요 메서드

```python
nums = [3, 1, 4, 1, 5, 9, 2]

nums.append(6)          # 맨 뒤에 추가 → [..., 6]
nums.insert(0, 0)       # 0번 위치에 0 삽입
nums.remove(1)          # 처음 나오는 1 제거
popped = nums.pop()     # 마지막 요소 꺼내기 (반환하면서 삭제)
nums.sort()             # 오름차순 정렬 (원본 변경)
nums.sort(reverse=True) # 내림차순 정렬
nums.reverse()          # 순서 뒤집기
print(len(nums))        # 길이
print(1 in nums)        # 포함 여부 확인 (True/False)
```

## 6. 튜플(Tuple)

리스트와 유사하지만 **변경 불가(immutable)**합니다. 소괄호 `()` 사용.

```python
point = (3, 5)
rgb = (255, 128, 0)

# 언패킹 (튜플의 강력한 기능)
x, y = point
print(x, y)  # 3 5

# 함수의 여러 반환값도 튜플로 처리
def min_max(lst):
    return min(lst), max(lst)

lo, hi = min_max([3, 1, 4, 1, 5, 9])
print(lo, hi)  # 1 9
```

## 7. 딕셔너리(Dictionary)

**Key: Value** 쌍으로 이루어진 자료형입니다. 중괄호 `{}` 사용.

```python
# 딕셔너리 생성
user = {
    'name': 'min oh',
    'age': 25,
    'department': '데이터팀',
    'skills': ['Python', 'SQL']
}

# 접근
print(user['name'])          # 'min oh'
print(user.get('age', 0))   # 25 (없으면 기본값 0 반환)

# 추가/수정/삭제
user['email'] = 'min@test.com'  # 추가
user['age'] = 26                 # 수정
del user['email']                # 삭제

# 순회
for key, value in user.items():
    print(f"{key}: {value}")

# 유용한 메서드
print(user.keys())    # 키 목록
print(user.values())  # 값 목록
print('name' in user) # 키 존재 여부 확인
```

## 8. 불(Boolean) 자료형

`True` 또는 `False` 두 가지 값만 가집니다.

```python
# 비교 연산자 → 불린 값 반환
print(10 > 5)    # True
print(10 == 10)  # True
print(10 != 5)   # True
print(10 >= 20)  # False

# 논리 연산자
print(True and False)  # False
print(True or False)   # True
print(not True)        # False

# False로 평가되는 값들 (Falsy Values)
# False, 0, "", [], {}, None
```

## 9. 자료형 변환

```python
# 형변환 함수
print(int("123"))     # "123" → 123 (문자열→정수)
print(float("3.14"))  # "3.14" → 3.14 (문자열→실수)
print(str(100))       # 100 → "100" (정수→문자열)
print(list((1,2,3)))  # (1,2,3) → [1,2,3] (튜플→리스트)

# input()은 항상 문자열!
age = int(input("나이를 입력하세요: "))  # 꼭 int()로 변환!
```

> [!IMPORTANT]
> `input()` 함수로 입력받은 값은 **항상 문자열(str)** 입니다. 숫자 계산에 쓰려면 반드시 `int()` 또는 `float()`로 변환하세요!

## 핵심 정리

| 자료형 | 특징 | 기호 | 변경 가능? |
|--------|------|------|-----------|
| `int`, `float` | 숫자 | - | O |
| `str` | 문자열 | `""`, `''` | X |
| `list` | 순서 있는 목록 | `[]` | O |
| `tuple` | 순서 있는 목록 | `()` | X |
| `dict` | 키-값 쌍 | `{}` | O |
| `bool` | 참/거짓 | - | - |
