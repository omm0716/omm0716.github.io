파이썬(Python)은 1991년 귀도 반 로섬이 개발한 인터프리터 언어로, 간결하고 직관적인 문법 덕분에 전 세계적으로 가장 인기 있는 프로그래밍 언어 중 하나입니다. 이번 포스트에서는 파이썬의 기초적인 특징과 주요 문법들을 정리해 봅니다.

---

## 1. 파이썬의 특징
- **휴머니즘 언어**: 인간이 생각하는 방식을 그대로 표현할 수 있을 정도로 직관적입니다.
- **무료 및 오픈 소스**: 누구나 언제 어디서든 무료로 다운로드하여 사용할 수 있습니다.
- **간결함**: 의도적으로 간결하게 설계되어 다른 사람의 코드를 이해하기 쉽습니다.

---

## 2. 변수와 숫자형 자료형

### 변수(Variable)
변수는 값을 저장하는 공간입니다. 식별자(이름)와 값을 지정하여 생성합니다.
```python
num = 10
print(num)  # 10 출력
```

### 숫자형 자료형
| 자료형 | 예시 | 설명 |
| :--- | :--- | :--- |
| **정수형 (int)** | 1, -5, 0 | 양/음의 정수 및 0 |
| **실수형 (float)** | 3.14, -1.5 | 소수점이 포함된 숫자 |
| **8진수 (oct)** | 0o177 | 0o 또는 0O로 시작 |
| **16진수 (hex)** | 0x8FF | 0x로 시작 |

---

## 3. 연산자 (Operators)

### 사칙연산 및 특수 연산자
- `+`, `-`, `*`, `/`: 더하기, 빼기, 곱하기, 나누기
- `**`: 거듭제곱 (예: `3 ** 4 = 81`)
- `%`: 나눗셈의 나머지 (예: `15 % 2 = 1`)
- `//`: 나눗셈의 몫 (예: `9 // 4 = 2`)

### 복합 대입 연산자
코드를 더 간결하게 만들어 줍니다.
- `x += 1` 은 `x = x + 1` 과 같습니다.
- `x -= 1`, `x *= 1`, `x /= 1` 등도 동일한 원리로 작동합니다.

---

## 4. 문자열(String)

### 문자열 만들기
파이썬에서는 큰따옴표(`"`)나 작은따옴표(`'`)를 3개씩 사용하여 여러 줄의 문자열을 만들 수도 있습니다.
```python
s1 = "Hello Python"
s2 = 'Python is fun'
s3 = """여러 줄의
문자열을 입력할 때 사용합니다."""
```

### 문자열 인덱싱과 슬라이싱
- **인덱싱**: `x[0]`은 첫 번째 문자를 가리킵니다.
- **슬라이싱**: `x[0:4]`는 0번부터 3번 인덱스까지의 문자를 잘라냅니다.

### 주요 문자열 함수
- `len(s)`: 문자열 길이 구하기
- `s.count('p')`: 특정 문자의 개수 세기
- `s.upper()` / `s.lower()`: 대문자 / 소문자 변환
- `s.strip()`: 양쪽 공백 제거
- `s.split(':')`: 특정 구분자로 문자열 나누기
- `s.replace('A', 'B')`: 문자열 치환

---

## 5. 리스트(List)

모든 자료형을 자유롭게 묶어서 사용할 수 있는 자료구조입니다. 대괄호(`[]`)를 사용합니다.

```python
a = [1, "Python", -0.5, [10, 20]]
```

### 리스트 관련 함수
- `append(x)`: 맨 뒤에 요소 추가
- `sort()`: 오름차순 정렬
- `reverse()`: 순서 뒤집기
- `index(x)`: 요소의 위치 찾기
- `insert(idx, x)`: 특정 위치에 요소 삽입
- `remove(x)`: 첫 번째로 나오는 특정 요소 제거
- `pop(idx)`: 특정 위치의 요소를 꺼내고 리스트에서 삭제

---

## 6. 튜플과 딕셔너리

### 튜플 (Tuple)
리스트와 비슷하지만, **한 번 생성하면 값을 변경하거나 삭제할 수 없습니다.** 소괄호(`()`)를 사용합니다.
```python
t1 = (1, 2, 3)
```

### 딕셔너리 (Dictionary)
Key와 Value의 쌍으로 이루어진 자료형입니다. 중괄호(`{}`)를 사용하며 순서가 없습니다.
```python
dic = {'name': 'min oh', 'age': 25}
print(dic['name'])  # 'min oh' 출력
```

---

## 7. 불(Bool) 자료형과 연산자

### 비교 연산자
`==`, `!=`, `>`, `<`, `>=`, `<=` 를 통해 `True` 또는 `False` 값을 얻습니다.

### 논리 연산자
- `and`: 둘 다 참이어야 참
- `or`: 하나만 참이어도 참
- `not`: 참을 거짓으로, 거짓을 참으로 반전
