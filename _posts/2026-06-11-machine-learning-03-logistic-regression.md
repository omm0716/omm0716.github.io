안녕하세요! 앞서 1강과 2강에서는 "점수", "무게"와 같은 **'연속적인 수치(숫자)'를 예측**하는 회귀(Regression) 분석에 대해 배웠습니다.

하지만 세상에는 숫자로 나타내기 어려운 문제들이 훨씬 많습니다.
- 이 이메일이 스팸일까? 아닐까? (스팸 분류)
- 이 환자의 검사 결과가 양성일까? 음성일까? (질병 예측)
- 이 점수로 합격할 수 있을까? 불합격일까? (합격 여부)

이렇게 **어떤 그룹(클래스)에 속하는지를 맞추는 문제를 분류(Classification) 문제**라고 합니다. 오늘 배울 **로지스틱 회귀(Logistic Regression)**는 이름에는 '회귀'가 들어가지만 실제로는 분류 작업에 가장 널리 쓰이는 아주 강력한 알고리즘입니다.

---

## 1. 선형 회귀로 분류를 하면 안 되는 이유

"공부 시간이 길면 합격, 짧으면 불합격"이라는 데이터를 선형 회귀로 학습시켰다고 가정해 봅시다. 
선형 회귀는 쭉 뻗어나가는 '직선'을 그리기 때문에, 공부 시간이 비정상적으로 긴 천재 학생 데이터가 하나라도 들어오면 직선의 기울기가 엉망으로 꺾여버립니다. 게다가 확률은 0%에서 100% 사이여야 하는데, 직선은 -200%, +500% 같은 이상한 값을 예측해버립니다.

### 해결책: 시그모이드(Sigmoid) 함수
이 직선을 어떻게든 0과 1(0% ~ 100%) 사이의 둥근 S자 모양 곡선으로 구부려버리는 수학적 마법이 바로 **시그모이드 함수**입니다.

> $S(x) = \frac{1}{1 + e^{-x}}$

공식이 복잡해 보이지만 기계가 다 계산해주니 걱정 마세요! 중요한 것은 **결과값이 무조건 0~1(0%~100%) 사이로 나온다**는 사실입니다. 만약 0.5(50%)를 넘으면 '합격(양성)', 넘지 못하면 '불합격(음성)'으로 판정하는 것이 로지스틱 회귀의 핵심 원리입니다.

---

## 2. 데이터 준비 및 모델 학습

scikit-learn을 사용하면 로지스틱 회귀 학습 과정도 이전 강좌들과 완벽하게 동일한 패턴으로 진행할 수 있습니다.

```python
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression

# 1. 데이터 불러오기 (예: 생존 여부, 합격 여부 등 0과 1로 된 Target 데이터)
data_df = pd.read_csv('./머신러닝실습용자료/합격여부데이터.csv', encoding='cp949')

# 2. Feature(X)와 Target(Y) 분리
data = data_df[['공부시간', '출석횟수']].to_numpy()
target = data_df['합격여부'].to_numpy() # 0: 불합격, 1: 합격

# 3. 데이터 분리 (8:2)
훈련용_data, 테스트용_data, 훈련용_target, 테스트용_target = train_test_split(
    data, target, test_size=0.2, random_state=42
)

# 4. 로지스틱 회귀 모델 생성 및 학습
lr = LogisticRegression()
lr.fit(훈련용_data, 훈련용_target)

print("분류 정확도:", lr.score(테스트용_data, 테스트용_target))
```

> [!NOTE]
> 여기서의 `score`는 회귀에서의 $R^2$ 점수가 아니라, **분류 정확도(Accuracy)**입니다. 즉, 테스트 세트 중 몇 개를 올바르게 합격/불합격으로 맞추었는지를 백분율(0~1)로 나타냅니다. 0.85가 나왔다면 85%의 확률로 정답을 맞췄다는 뜻입니다.

---

## 3. 확률을 직접 눈으로 확인하기

분류 모델은 단순히 "합격입니다"라고 답하는 것을 넘어, **"몇 퍼센트의 확률로 합격입니까?"**라는 질문에도 답할 수 있어야 합니다. `predict_proba` (Predict Probability) 함수를 사용하면 각 클래스에 대한 확률을 소수점 형태로 확인할 수 있습니다.

```python
# 상위 5개 데이터에 대해 합격/불합격 확률 예측
probabilities = lr.predict_proba(테스트용_data[:5])

print(probabilities)
```

**출력 예시:**
```text
[[0.82  0.18]
 [0.11  0.89]
 [0.95  0.05]
 [0.45  0.55]
 [0.20  0.80]]
```

결과가 2차원 배열로 나옵니다. 
첫 번째 열은 `0(불합격)`일 확률, 두 번째 열은 `1(합격)`일 확률입니다.
- 첫 번째 학생은 불합격 확률 82%, 합격 확률 18%이므로 기계는 **불합격**으로 예측합니다.
- 네 번째 학생은 합격 확률이 55%로 아슬아슬하게 **합격**을 예측하게 됩니다.

## 💡 요약

| 개념 | 설명 |
|---|---|
| **분류 (Classification)** | 데이터를 범주(합격/불합격, 개/고양이 등)로 나누는 예측 문제 |
| **로지스틱 회귀** | 선형 회귀 방식을 기반으로 데이터를 분류하는 가장 기초적인 분류 알고리즘 |
| **시그모이드 함수 (Sigmoid)** | 어떤 값이 들어가든 무조건 0과 1 사이의 실수(확률)로 변환해주는 마법의 함수 |
| **predict_proba** | 단순히 0과 1로 답을 주는 것을 넘어, 각 클래스에 속할 확률을 수치로 알려주는 함수 |

지금까지 수학적인 선(또는 곡선)을 그어 데이터를 예측하고 분류하는 알고리즘들을 살펴보았습니다. 다음 시간에는 선을 긋는 것이 아니라, "내 주변에 누가 제일 많은가?"라는 다수결의 원리로 동작하는 직관적인 알고리즘인 **KNN(K-Nearest Neighbors)**에 대해 알아보겠습니다.
