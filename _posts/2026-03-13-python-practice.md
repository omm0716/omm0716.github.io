## 문제 1. 제곱 계산하기
**Q: `input()` 함수를 이용해 숫자 2개를 입력받아 각각의 제곱을 계산하여 출력하세요.**
- 입력 예시: a=3, b=5

```python
a = int(input("a의 값을 입력하세요: "))
b = int(input("b의 값을 입력하세요: "))

print(a ** 2)
print(b ** 2)
```

> **설명:** `input()`으로 입력받은 값은 문자열이므로, 계산을 위해 `int()`를 사용해 정수형으로 변환해주어야 합니다.

## 문제 2. 문자열 슬라이싱 활용 (날짜/날씨)
**Q: '20241014sunny' 텍스트에서 연도, 월, 일, 날씨를 각각 추출하여 변수에 저장하고 출력하세요.**

```python
text = '20241014sunny'

year = text[0:4]
month = text[4:6]
day = text[6:8]
weather = text[8:]

print(year + "=year")
print(month + "=month")
print(day + "=day")
print(weather + "=weather")
```

## 문제 3. 평균 점수 계산하기
**Q: 국어(80), 영어(75), 수학(55) 점수의 합계와 평균을 구하여 출력하세요.**

```python
kor = 80
eng = 75
math = 55

average = (kor + eng + math) / 3
print(average)
```

## 문제 4. 리스트 정렬하기
**Q: 리스트 `[1, 3, 5, 4, 2]`를 오름차순과 내림차순으로 각각 정렬하여 출력하세요.**

```python
a = [1, 3, 5, 4, 2]

# 오름차순
a.sort()
print(a)

# 내림차순 (정렬된 상태에서 순서를 뒤집음)
a.reverse()
print(a)
```

## 문제 5. 주민등록번호 분리하기
**Q: 주민등록번호 '881120-1068234'에서 생년월일(앞자리)과 뒷자리 번호를 나누어 출력하세요.**

## 문제 7. 문자열 치환
**Q: 문자열 'a:b:c:d'에서 `:`을 `#`으로 바꾸어 출력하세요.**

```python
text = 'a:b:c:d'
new_text = text.replace(':', '#')
print(new_text)
```

## 문제 9. 리스트 중복 제거 (응용)
**Q: 리스트 `[1, 1, 1, 2, 2, 3, 3, 3, 4, 4, 5]`에서 중복된 숫자를 제거하여 출력하세요.**

```python
a = [1, 1, 1, 2, 2, 3, 3, 3, 4, 4, 5]

# set(집합) 자료형을 사용하면 중복을 아주 쉽게 제거할 수 있습니다.
result = list(set(a))
print(result)
```
