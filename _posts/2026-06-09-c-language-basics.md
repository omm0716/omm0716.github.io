C언어(C Language)는 1972년 벨 연구소의 데니스 리치(Dennis Ritchie)가 개발한 **범용 프로그래밍 언어**입니다. 운영체제, 임베디드 시스템, 게임 엔진 등 성능이 중요한 곳에서 여전히 광범위하게 사용됩니다. 이번 시리즈에서는 Visual Studio 환경에서 C언어의 핵심 문법을 실습 코드 중심으로 배워봅니다.

## 1. C언어의 특징

| 특징 | 설명 |
|------|------|
| **절차 지향** | 위에서 아래로 순서대로 실행되는 구조 |
| **컴파일 언어** | 소스 코드 전체를 기계어로 변환 후 실행 → 속도 빠름 |

| **메모리 직접 관리** | 포인터를 이용해 메모리를 직접 제어 가능 |
| **이식성** | 다양한 플랫폼에서 동일한 코드 실행 가능 |

## 2. 첫 번째 C 프로그램 구조

모든 C 프로그램은 `main()` 함수에서 시작합니다.

```c
#include <stdio.h>   // 표준 입출력 헤더 포함

int main()           // 프로그램 진입점
{
    printf("Hello World!\n");  // 화면에 출력
    return 0;                  // 정상 종료 신호
}
```

| 구성 요소 | 설명 |
|-----------|------|
| `#include <stdio.h>` | `printf`, `scanf` 등 표준 입출력 함수를 사용하기 위해 필요 |
| `int main()` | 프로그램이 시작되는 함수. 반드시 존재해야 함 |
| `{ }` | 함수의 시작과 끝을 나타내는 중괄호 |
| `return 0;` | 운영체제에 프로그램이 정상 종료됐음을 알림 |

> [!NOTE]
> Visual Studio에서 `scanf` 함수 사용 시 보안 경고가 발생합니다. 이를 방지하기 위해 파일 최상단에 `#define _CRT_SECURE_NO_WARNINGS`를 추가합니다.

## 3. 변수(Variable)와 자료형

변수는 데이터를 저장하는 **이름 붙은 메모리 공간**입니다. C언어는 반드시 사용 전에 **자료형과 함께 선언**해야 합니다.

### 주요 자료형

| 자료형 | 크기 | 범위 | 예시 |
|--------|------|------|------|
| `int` | 4 byte | -2,147,483,648 ~ 2,147,483,647 | 나이, 개수, 점수 |
| `float` | 4 byte | 소수점 약 6자리 | 온도, 비율 |
| `double` | 8 byte | 소수점 약 15자리 | 정밀한 계산 |
| `char` | 1 byte | 문자 1개 (ASCII) | 알파벳, 숫자 문자 |

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main()
{
    int age = 25;          // 정수형 변수
    float height = 175.5;  // 단정밀도 실수
    char grade = 'A';      // 문자형 변수

    printf("나이: %d\n", age);
    printf("키: %.1f cm\n", height);
    printf("등급: %c\n", grade);

    return 0;
}
```

## 4. printf와 scanf - 표준 입출력

### printf: 화면에 출력

`printf("형식 문자열", 변수1, 변수2, ...);` 형태로 사용합니다.

| 서식 지정자 | 의미 | 예시 |
|-------------|------|------|
| `%d` | 정수(int) 출력 | `printf("%d", 42)` |
| `%f` | 실수(float/double) 출력 | `printf("%.2f", 3.14)` |
| `%c` | 문자(char) 출력 | `printf("%c", 'A')` |
| `%s` | 문자열 출력 | `printf("%s", "hello")` |
| `\n` | 줄 바꿈(개행) | - |

### scanf: 키보드 입력 받기

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main()
{
    int a = 0;
    int b = 0;

    printf("두 정수를 공백으로 구분해서 입력하세요.\n");
    scanf("%d%d", &a, &b);  // & : 변수의 주소

    int c1 = a * b;  // 곱
    int c2 = a + b;  // 합
    int c3 = a - b;  // 차

    printf("곱: %d\n 합: %d\n 차: %d", c1, c2, c3);

    return 0;
}
```

> [!IMPORTANT]
> `scanf`에서 변수 앞에 `&`(주소 연산자)를 반드시 붙여야 합니다. `&a`는 "변수 a가 저장된 메모리 주소"를 의미합니다.

## 5. 산술 연산자

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main()
{
    int m = 127670;  // 금액

    // 각 권종의 장수 계산 (정수 나눗셈 + 나머지 활용)
    int m50000 = m / 50000;
    int m10000 = (m % 50000) / 10000;
    int m5000  = (m % 10000) / 5000;
    int m1000  = (m % 5000)  / 1000;
    int m500   = (m % 1000)  / 500;
    int m100   = (m % 500)   / 100;
    int m50    = (m % 100)   / 50;
    int m10    = (m % 50)    / 10;

    int sum = m50000 + m10000 + m5000 + m1000 + m500 + m100 + m50 + m10;
    printf("총 지폐/동전 장수: %d", sum);

    return 0;
}
```

| 연산자 | 의미 | 예시 | 결과 |
|--------|------|------|------|
| `+` | 덧셈 | `10 + 3` | 13 |
| `-` | 뺄셈 | `10 - 3` | 7 |
| `*` | 곱셈 | `10 * 3` | 30 |
| `/` | 나눗셈(몫) | `10 / 3` | **3** (정수) |
| `%` | 나머지 | `10 % 3` | 1 |

> [!TIP]
> C언어에서 **정수 ÷ 정수 = 정수**입니다. `10 / 3`의 결과는 `3.333...`이 아니라 `3`입니다. 소수점 이하는 버려집니다.

## 6. 초(秒)를 시:분으로 변환하기

나머지 연산(`%`)과 나눗셈(`/`)을 결합한 실용적인 예제입니다.

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main()
{
    int sec;
    int min;
    int hour;

    scanf("%d", &sec);   // 총 초를 입력받음

    hour = sec / 3600;          // 1시간 = 3600초
    min  = (sec % 3600) / 60;   // 나머지 초에서 분 추출

    printf("%d hour %d min", hour, min);

    return 0;
}
```

**실행 예시:**
- 입력: `7384`
- 출력: `2 hour 3 min`

## 핵심 정리

| 개념 | 내용 |
|------|------|
| **헤더** | `#include <stdio.h>` - printf/scanf 사용을 위해 필수 |
| **main 함수** | 모든 C 프로그램의 시작점 |
| **변수 선언** | `자료형 변수명 = 초기값;` (예: `int age = 0;`) |
| **printf** | `printf("형식", 변수)` - 서식 지정자(%d, %f, %c, %s) 사용 |
| **scanf** | `scanf("형식", &변수)` - 변수 앞에 반드시 `&` 붙이기 |
| **정수 나눗셈** | 결과가 항상 정수 (소수점 버림), 나머지는 `%` 사용 |

다음 강에서는 **조건문(if, 삼항 연산자, switch)**으로 프로그램에 판단 기능을 추가해봅니다!
